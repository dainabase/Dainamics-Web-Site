# Calculateur ROI IA : Estimez votre Potentiel

Avant d'investir dans l'IA, une question s'impose : quel retour puis-je espérer ? Ce guide vous donne une méthodologie rigoureuse pour calculer le ROI de vos projets IA, avec des benchmarks issus de projets réels.

## Pourquoi calculer le ROI avant de se lancer ?

### Les chiffres du marché

Selon les études 2025 sur l'adoption IA en PME :

- **ROI moyen** : 187% sur 3 ans (source : McKinsey SME Report)
- **Délai breakeven** : 7 mois en moyenne
- **Retour par dollar investi** : 3.70$ (projets bien exécutés)
- **Taux d'échec** : 70% des projets n'atteignent pas leurs objectifs

Cette dernière statistique souligne l'importance d'un calcul ROI rigoureux AVANT le projet.

## Formule de base du ROI

### La formule simple

```
ROI = (Gains - Coûts) / Coûts × 100
```

### La formule complète pour projets IA

```
ROI IA = (Gains directs + Gains indirects - Coûts totaux) / Coûts totaux × 100

Où :
- Gains directs = Temps économisé × Coût horaire + Revenus additionnels
- Gains indirects = Réduction erreurs + Satisfaction client + Opportunités
- Coûts totaux = Développement + Déploiement + Formation + Maintenance + API
```

## Calculer les gains directs

### 1. Temps économisé

**Formule** :
```
Gain temps = Heures économisées/mois × Coût horaire × 12 mois
```

**Exemple — Automatisation facturation** :

| Élément | Avant | Après | Économie |
|---------|-------|-------|----------|
| Création factures | 8h/mois | 0.5h | 7.5h |
| Envoi et suivi | 4h/mois | 0h | 4h |
| Relances | 6h/mois | 0.5h | 5.5h |
| **Total** | 18h/mois | 1h | **17h/mois** |

**Calcul** : 17h × 65 CHF × 12 = **13'260 CHF/an**

### 2. Réduction des coûts opérationnels

**Formule** :
```
Gain coûts = (Coût process actuel - Coût process IA) × Volume × 12
```

**Exemple — Support client chatbot** :

| Métrique | Avant | Après |
|----------|-------|-------|
| Tickets/mois | 500 | 500 |
| Coût moyen/ticket | 15 CHF | 15 CHF |
| Tickets traités IA | 0% | 70% |
| Tickets humains | 500 | 150 |

**Calcul** : 
- Avant : 500 × 15 CHF = 7'500 CHF/mois
- Après : 150 × 15 CHF + 50 CHF (API IA) = 2'300 CHF/mois
- Économie : 5'200 CHF/mois = **62'400 CHF/an**

### 3. Revenus additionnels

**Sources potentielles** :
- Augmentation conversion (meilleur suivi leads)
- Upselling automatisé
- Réduction churn (support proactif)
- Nouveaux services possibles

**Exemple — IA qualification leads** :

| Métrique | Avant | Après | Amélioration |
|----------|-------|-------|--------------|
| Leads/mois | 100 | 100 | — |
| Taux conversion | 5% | 8% | +60% |
| Valeur moyenne | 5'000 CHF | 5'000 CHF | — |
| CA mensuel | 25'000 CHF | 40'000 CHF | +15'000 CHF |

**Calcul** : 15'000 CHF × 12 = **180'000 CHF/an** (revenus additionnels)

## Calculer les gains indirects

### 1. Réduction des erreurs

**Formule** :
```
Gain erreurs = Taux erreur actuel × Volume × Coût moyen erreur × Réduction attendue
```

**Exemple — Extraction documents** :

- Erreurs actuelles : 5% sur 200 documents/mois = 10 erreurs
- Coût moyen erreur : 150 CHF (correction + impact)
- Réduction avec IA : 90%
- Économie : 10 × 150 × 90% = 1'350 CHF/mois = **16'200 CHF/an**

### 2. Amélioration satisfaction client

Plus difficile à quantifier, mais réel :

| Impact | Estimation valeur |
|--------|-------------------|
| Réduction churn 1% | Valeur client × Nb clients × 1% |
| NPS +10 points | ~5% revenus additionnels |
| Temps réponse -50% | Réduction tickets escalade |

### 3. Opportunités stratégiques

- Capacité à scaler sans embaucher
- Données pour meilleures décisions
- Avantage compétitif

**Règle pratique** : Estimez les gains indirects à 20-30% des gains directs.

## Exemples de calculs ROI complets

### Exemple 1 : Chatbot support client

**Contexte** : PME 30 employés, 400 tickets support/mois

**Investissement** :
- Développement : 12'000 CHF
- Formation : 1'500 CHF
- **Total initial** : 13'500 CHF

**Coûts récurrents** :
- API + hébergement : 150 CHF/mois
- Maintenance : 300 CHF/mois
- **Total annuel** : 5'400 CHF

**Gains** :
- 70% tickets automatisés : 280 tickets × 12 CHF économisés = 3'360 CHF/mois
- Disponibilité 24/7 : estimation +1'000 CHF/mois (satisfaction)
- **Total annuel** : 52'320 CHF

**ROI Année 1** :
```
ROI = (52'320 - 13'500 - 5'400) / (13'500 + 5'400) × 100
ROI = 33'420 / 18'900 × 100 = 177%
```

**Délai breakeven** : 4 mois

### Exemple 2 : Automatisation facturation

**Contexte** : Fiduciaire, 150 clients, facturation mensuelle

**Investissement** :
- Développement : 8'000 CHF
- Intégration Bexio : 2'000 CHF
- Formation : 1'000 CHF
- **Total initial** : 11'000 CHF

**Coûts récurrents** :
- Outils automation : 50 CHF/mois
- Maintenance : 200 CHF/mois
- **Total annuel** : 3'000 CHF

**Gains** :
- Temps économisé : 15h/mois × 80 CHF = 1'200 CHF/mois
- Réduction erreurs : 200 CHF/mois
- Paiements plus rapides (-5 jours) : 300 CHF/mois (trésorerie)
- **Total annuel** : 20'400 CHF

**ROI Année 1** :
```
ROI = (20'400 - 11'000 - 3'000) / (11'000 + 3'000) × 100
ROI = 6'400 / 14'000 × 100 = 46%
```

**ROI Année 2** (sans coût initial) :
```
ROI = (20'400 - 3'000) / 3'000 × 100 = 580%
```

**Délai breakeven** : 8 mois

### Exemple 3 : IA extraction documents

**Contexte** : Entreprise import/export, 300 documents/mois

**Investissement** :
- Développement : 18'000 CHF
- OCR + training : 4'000 CHF
- Intégration ERP : 3'000 CHF

**Coûts récurrents** :
- API Vision : 100 CHF/mois
- Hébergement : 100 CHF/mois
- Maintenance : 400 CHF/mois
- **Total annuel** : 7'200 CHF

**Gains** :
- Temps saisie : 25h/mois × 50 CHF = 1'250 CHF/mois
- Réduction erreurs : 500 CHF/mois
- Traitement accéléré : 400 CHF/mois

**ROI Année 1** :
```
ROI = (25'800 - 25'000 - 7'200) / (25'000 + 7'200) × 100
ROI = -6'400 / 32'200 × 100 = -20%
```

**ROI Année 2** :
```
ROI = (25'800 - 7'200) / 7'200 × 100 = 258%
```

**Délai breakeven** : 16 mois

## Benchmarks par type de projet

### Références sectorielles

| Type projet | ROI moyen An 1 | Breakeven | Risque |
|-------------|----------------|-----------|--------|
| Chatbot support | 150-250% | 3-6 mois | Faible |
| Automatisation admin | 80-150% | 6-9 mois | Faible |
| Extraction documents | 50-100% | 9-15 mois | Moyen |
| IA prédictive | 100-300% | 12-18 mois | Élevé |
| Plateforme custom | Variable | 18-24 mois | Élevé |

### Facteurs de succès

**Projets à ROI élevé** :
- Processus répétitif clairement défini
- Volume suffisant (>100 occurrences/mois)
- Données de qualité disponibles
- Sponsor interne engagé
- Objectifs mesurables définis

**Projets à risque** :
- Processus mal défini
- Données dispersées ou absentes
- Pas de métrique de succès claire
- Résistance au changement
- Scope trop ambitieux

## Outils de calcul

### Calculateur simplifié

**Entrées requises** :
1. Heures économisées par mois
2. Coût horaire moyen
3. Coût du projet
4. Coûts récurrents mensuels

**Formule rapide** :
```
Breakeven (mois) = Coût projet / (Heures × Coût horaire - Coûts récurrents)
ROI An 1 = ((Heures × Coût horaire × 12) - Projet - (Récurrent × 12)) / (Projet + Récurrent × 12)
```
